# Changelog

## [0.1.0] - 2025-02-11

### Added

- **Generate and Validate EPUB** command — validate expanded EPUB and generate `.epub` file
- **Generate EPUB Only** command — generate `.epub` without detailed validation
- **Generate EPUB and Validation Report** command — validate and export report (Markdown/Text/JSON)
- **Unzip EPUB and Validate** command — extract `.epub` files and validate contents
- Problems panel integration with inline error squiggles
- Status bar indicator showing validation state
- Auto-detection of EPUB projects via `mimetype` file
- Multiple EPUB project support with Quick Pick selection
- Explorer context menu integration for folders and `.epub` files
- File browser for selecting `epubcheck.jar` and report directory
- First-launch prompt to download EPUBCheck
